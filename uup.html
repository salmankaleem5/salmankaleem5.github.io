University User Profiles

Languages and Technologies: PHP, JavaScript, SQL, WordPress

Background

University User Profiles is a system for departments of the University of Connecticut to manage, customize, and display profiles of their faculty and staff. This system was built into the university’s content management system and was integrated with the university’s user management system. This system contains user accounts for all faculty, staff, and students with information like name and email-address. The goal of this project was to make a simplified and easy process for website administrators to show these profiles on their existing websites without having to manually create each profile. Thanks to the user management system integration, this feature saved departments a lot of time.

Components
  • Backend user interface for website administrators to import faculty and staff credentials from the university’s user management system
  • Backend user interface for website administrators to customize profiles
  • Customizable front-end templates to display a list or table profiles based on custom parameters
  • Customizable front-end template to display individual profiles with extra content
